 Мне нужно разработать АИ агента, который будет работать в бесплатном аккаунте Cloudflare Workers и использовать Gemini API с моделью Gemini-2.5-Flash-Lite
ОЧЕНЬ ВАЖНО: ОБЯЗАТЕЛЬНО ПРОВЕРЯЙ ЧТОБЫ ИСПОЛЬЗОВАЛАСЬ ТОЛЬКО ЭТА МОДЕЛЬ В КОДЕ И НИКАКАЯ ДРУГАЯ

Тебе нужно сначала продумать оптимальную архитектуру этого агента и составить [Agents.md](http://Agents.md) файл c его описанием, процессом разработки, тестами и всем что нужно для оптимальной разработки такого агента. Тебе нужно делать все действия с Cloduflare через CLI интерфейс, включая создание агента, базы данных для него и деплоя.

Я предоствлю все API ключи и доступ к телеграм боту когда тебе нужно

Как должен работать этот агент. Каждый день в 23:55 по Московскому времени он должен зайти на страницу https://liveclasses.ru/schedule/ (первый раз он должен это сделать сейчас когда создаем этого агента) найти раздел **ЗАВТРА /число и месяц/** и собрать данные о каждом предстоящей Завтра трансляции в базу данных: название трансляции, время начала и автор.
Затем, агент должен составить план уведомлений пользователя и за 15 минут до начала каждой трансляции писать в телеграм бот пользователя и на русском сообщать что через 15 минут начнется мастер класс - писать его назание, время начала и ссылку на трансляцию

Так же пользователь может спросить что-то у этого агента, например, какая ближайшая трансляция и агент должен использовать данные из этой таблицы

Перед обновлением таблицы агент всегда должен сначала удалять прошлые данные. То есть в таблицы всегда должны быть только данные о трансляция на завтра, данные о прошлых трансляциях должны удаляться перед добавлением новых

Если пользователь отправляет слово “тест” то бот должен ответить со случайным напоминанием о ближайших трансляциях
